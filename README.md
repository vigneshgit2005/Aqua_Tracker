
# 💧 Aqua Tracker

Aqua Tracker is a smart and minimal water intake tracking application designed to help users stay hydrated by monitoring daily water consumption, maintaining streaks, and visualizing average intake patterns.

---

## 🚀 Features

* 📊 **Daily Water Tracking**
  Log your daily water intake easily with a clean and intuitive interface.

* 🔥 **Streak System**
  Stay consistent! Track how many consecutive days you meet your hydration goals.

* 📈 **Average Intake Monitoring**
  Analyze your hydration habits with calculated daily and weekly averages.

* 🎯 **Custom Goals**
  Set personalized daily water intake targets based on your needs.

* 📅 **History Tracking**
  View past records and monitor your progress over time.

* 📱 **Responsive UI**
  Works smoothly across desktop and mobile devices.

---

## 🧠 Project Idea

Aqua Tracker aims to solve a common health issue — **inconsistent hydration habits**. By combining streak motivation and data visualization, the app encourages users to build a sustainable water-drinking routine.

---

## 🛠️ Tech Stack

* **Frontend:** React.js / Streamlit (depending on your version)
* **Styling:** Tailwind CSS
* **Backend (optional):** Node.js / Python
* **Storage:** Local Storage / Database (Firebase / MongoDB if used)

---

## 📊 How It Works

1. User logs daily water intake (in liters/ml)
2. App updates:

   * Daily progress
   * Streak count
   * Average intake
3. Visual insights help users improve consistency
